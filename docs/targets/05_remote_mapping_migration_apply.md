# 目标 05：远程源映射安全应用与复检

状态：已完成（1.0.12 / versionCode 176）

## 目标

在已确认的远程源映射预览基础上，安全更新本地媒体库路径并记录迁移任务；无法可靠映射的条目保留原记录并引导定向重扫。

## 范围

- 应用已确认的路径重写，仅修改应用内部媒体库与相关刮削记录。
- 写入可追踪的迁移任务结果，包含成功、跳过和待重扫数量。
- 应用后执行定向诊断/复检，并展示结果。
- 为原子写入、跳过策略和结果汇总补充单测。

## 验收

1. 不删除远程文件、远程配置或未能映射的媒体记录。
2. 中途失败不会留下半写入媒体库。
3. 应用完成后用户能看到迁移结果和下一步复检入口。

## 完成记录

- 已确认的映射配置通过既有远程配置保存路径落盘，不重写媒体库路径。
- 映射变更保存后创建独立的定向源健康复检任务，只探测变更的远程源。
- 复检任务记录成功、需处理或失败结果，并刷新主页、搜索和来源列表状态。
