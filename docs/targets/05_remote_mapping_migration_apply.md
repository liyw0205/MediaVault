# 目标 05：远程源映射安全应用与复检

状态：已完成（1.0.12 / versionCode 176）

## 目标

在已确认的远程源映射预览基础上，安全更新远程连接配置并执行定向复检；媒体库路径保持稳定，避免未经验证的路径重写。

## 范围

- 应用已确认的远程连接配置更新，不修改媒体库路径或刮削记录。
- 写入可追踪的定向复检任务结果。
- 应用后执行定向诊断/复检，并展示结果。
- 复用配置原子写入，并为预览策略和结果汇总补充单测。

## 验收

1. 不删除远程文件、远程配置或未能映射的媒体记录。
2. 中途失败不会改写或部分改写媒体库路径。
3. 应用完成后用户能看到迁移结果和下一步复检入口。

## 完成记录

- 已确认的映射配置通过既有远程配置保存路径落盘，不重写媒体库路径。
- 映射变更保存后创建独立的定向源健康复检任务，只探测变更的远程源。
- 复检任务记录成功、需处理或失败结果，并刷新主页、搜索和来源列表状态。
