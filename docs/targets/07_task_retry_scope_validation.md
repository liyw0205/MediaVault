# 目标 07：任务重试前范围有效性校验

状态：已完成（1.0.14 / versionCode 178）

## 目标

在按原任务范围重试前确认相关媒体源仍然存在，避免已移除目录或远程源形成必然失败的新任务。

## 范围

- 全库重试要求当前至少存在一个已配置媒体源。
- 局部重试逐项核对原本机目录 URI 和远程源 id。
- 来源缺失时不启动后台任务，并展示缺失类型、数量和恢复建议。
- 为全库、局部、混合范围和失效范围增加 JVM 单测。

## 验收

1. 已移除来源不会创建新的扫描任务。
2. 仍有效的局部范围保持原参数重放。
3. 阻止原因不读取或展示远程凭据。

## 完成记录

- 新增纯逻辑范围验证结果，报告缺失本机目录和远程源。
- 任务详情在启动重试前读取当前源配置并校验，失败时只展示缺失计数。
- JVM 单测覆盖无源全库、有效局部、混合缺失、单类型范围和不兼容范围。
