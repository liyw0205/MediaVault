# 目标 03：可解释失败重试与最近错误

状态：待开始

## 目标

为失败、部分成功和取消任务提供可理解的最近错误摘要与可执行的重试路径，避免用户只能看到笼统失败状态。

## 范围

- 持久化受控、脱敏的最近错误摘要和失败类别。
- 任务详情展示失败原因、影响范围和建议动作。
- 支持只对可安全重放的扫描/刮削任务生成重试请求；不可重试任务明确说明原因。
- 为失败分类、脱敏和重试资格补充单测。

## 验收

1. 失败任务能显示最近一次可读错误，不泄漏远程密码或令牌。
2. 可重试任务可以恢复原有安全范围；不可重试任务没有误导性入口。
3. 重试结果会作为新任务记录，不覆盖原任务证据。
