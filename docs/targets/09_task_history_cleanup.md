# 目标 09：任务历史分级清理

状态：已完成（1.0.16 / versionCode 180）

## 目标

让用户释放无用任务历史时可以保留失败证据和待处理上下文，避免“清空已完成”一次删除所有排障记录。

## 范围

- 支持仅清理成功任务。
- 保留清理所有已结束任务的既有能力。
- 运行中任务在任一策略下都不会删除。
- “可继续处理”筛选同时包含关联问题和可安全重试任务。

## 验收

1. 仅清理成功不会删除失败、部分成功、取消或运行中任务。
2. 清理全部已结束仍保留运行中任务。
3. 清理后摘要、筛选和首页最近任务可刷新。

## 完成记录

- 新增任务历史清理策略与 removed/kept 统计结果。
- 任务中心提供“仅清理成功任务”和“清空已完成”两个明确入口。
- JVM 单测覆盖两种策略、运行中保护、顺序保持和空列表统计。
