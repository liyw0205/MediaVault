# 目标 04：远程源映射变更影响预览

状态：已完成（1.0.11 / versionCode 175）

## 目标

在修改远程源协议、host、端口或 basePath 前，明确展示实际变更字段与关联媒体记录，并让用户确认更新连接配置后定向复检。

## 范围

- 计算映射字段变化、按远程源 id 关联的媒体数量和路径样本。
- 配置编辑保存前展示只读预览，不写入数据。
- 明确本功能不会探测、删除或修改远程源文件。
- 为字段变化和预览汇总补充 JVM 单测。

## 验收

1. host、端口或 basePath 变更时必定先展示影响预览。
2. 预览能区分无映射变化与需确认变化，并列出实际变更字段。
3. 取消预览不会写入远程配置或媒体库。

## 完成记录

- 新增纯逻辑预览，按远程配置 id 精确计算关联媒体条目和展示样本。
- 修改协议、主机、端口或 basePath 时，编辑保存后先显示预览确认。
- 确认更新只写入远程连接配置；取消不会更改内存配置、落盘配置或媒体库。
