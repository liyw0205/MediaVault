# 目标 04：远程源映射变更影响预览

状态：已完成（1.0.11 / versionCode 175）

## 目标

在修改远程源 host、端口或 basePath 前，明确展示将影响的媒体库条目与刮削记录，并让用户选择只更新库映射或保留原配置后重扫。

## 范围

- 计算旧/新远程映射下的受影响条目数、路径样本和不可映射条目。
- 配置编辑保存前展示只读预览，不写入数据。
- 明确本功能不会探测、删除或修改远程源文件。
- 为路径映射和预览汇总补充 JVM 单测。

## 验收

1. host、端口或 basePath 变更时必定先展示影响预览。
2. 预览能区分可重写、需要重扫和无影响条目。
3. 取消预览不会写入远程配置或媒体库。

## 完成记录

- 新增纯逻辑预览，按远程配置 id 精确计算关联媒体条目和展示样本。
- 修改协议、主机、端口或 basePath 时，编辑保存后先显示预览确认。
- 确认更新只写入远程连接配置；取消不会更改内存配置、落盘配置或媒体库。
