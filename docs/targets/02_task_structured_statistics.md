# 目标 02：任务中心扫描与刮削结构化统计

状态：待开始

## 目标

让扫描和刮削任务的结果能直接解释处理范围、成功量、跳过量和问题量，减少用户从任务详情再回到维护中心确认结果的成本。

## 范围

- 为扫描、增量刮削、重刮等任务记录结构化统计字段。
- 任务详情按一致格式展示本机/远程范围、发现条目、写入条目、跳过条目、问题条目和耗时。
- 保持旧任务记录可兼容读取，并为缺失统计提供清晰降级文案。
- 为序列化、汇总和兼容读取补充 JVM 单测。

## 验收

1. 新任务详情不依赖拼接字符串即可展示核心统计。
2. 旧版 `library-tasks.json` 可正常读取。
3. 统计字段与任务实际结果一致，失败或取消任务不伪装为成功。
