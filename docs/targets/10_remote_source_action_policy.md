# 目标 10：远程源操作资格统一

状态：已完成（1.0.17 / versionCode 181）

## 目标

让远程源管理中的操作入口与当前源状态一致，避免无媒体、无问题或缺凭据时启动无意义流程。

## 范围

- 无关联媒体时禁用“查看媒体”和“重刮此源”。
- 无关联问题时禁用“查看问题”。
- 缺凭据时保留复检和补凭据入口，但禁止重刮。
- 健康状态不阻止用户主动复检或对已有媒体执行重刮。

## 验收

1. 每个按钮资格只由统一策略计算，不再散落判断。
2. 被阻止的重刮不会创建任务，并能说明缺凭据或无媒体。
3. 有媒体且凭据完整的远程源可正常重刮。

## 完成记录

- 新增纯逻辑远程源操作策略和明确的重刮阻止原因。
- 远程源列表统一使用策略控制四类操作。
- JVM 单测覆盖空源、无问题、缺凭据、正常源和健康状态组合。
